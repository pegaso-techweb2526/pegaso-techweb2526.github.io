# Introduzione del corso
Il laboratorio si concentra sull'uso di [CSS (Cascading Style Sheets)](https://it.wikipedia.org/wiki/CSS) per progettare e stilizzare una pagina web, con l'obiettivo di creare una home page funzionale e visivamente accattivante per il sito web del corso di laurea in Informatica. L'attività ha come obiettivo primario quello di applicare le competenze teoriche apprese in aula alla realizzazione di un progetto concreto. Durante il laboratorio, ogni studente avrà la possibilità di esplorare e implementare le principali tecniche di stilizzazione CSS, inclusi layout responsivi, gestione dei colori, tipografia, e la creazione di interfacce utente intuitive.  
Per facilitare il lavoro, il sito web avrà una struttura base già definita, che includerà sezioni come la descrizione del corso di laurea, gli obiettivi formativi, le informazioni sul corpo docente, e le modalità di accesso al programma. Gli studenti dovranno concentrarsi principalmente sulla personalizzazione visiva, utilizzando CSS per migliorare l'estetica e l'usabilità della pagina. Durante il laboratorio, ci sarà anche spazio per esplorare le migliori pratiche di design, come l'accessibilità, la coerenza visiva e la compatibilità cross-browser.  
Questo progetto non solo aiuterà gli studenti a consolidare le proprie competenze CSS, ma offrirà anche un'esperienza pratica nell'affrontare i problemi quotidiani di progettazione web, come la gestione di layout complessi, l'ottimizzazione per dispositivi mobili, e la creazione di design adattabili a diverse risoluzioni. Al termine del laboratorio, ciascun partecipante avrà una solida base su cui poter costruire e migliorare le proprie capacità di sviluppo front-end per il web.

# Argomenti del corso
Segue una breve lista dei principali argomenti trattati nel corso.
- Elementi base di CSS: principi base per la stilizzazione di un sito
- Elementi avanzati di CSS: contenuti avanzati per stilizzazioni più complesse
- Expert CSS: funzionalità complesse per stilizzazioni di alto livello
